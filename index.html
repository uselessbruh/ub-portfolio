Achievements

  1. FOSS Fest Winter Hackathon 2025 Winner

    2025 • €1000 Prize

    Part of the winning team at FOSS Fest Winter Hackathon 2025, competing against more than 500 teams worldwide in this Amsterdam-based online event. During the hackathon, our team developed “Enhanced TELEports”, a modified Telegram application for Ubuntu Touch that introduced additional features and performance improvements to the platform. I served as the main backend developer, leading the backend development and ensuring smooth integration and efficient mediator logic.

    Project Details:
    Enhanced TELEports is built with the Telegram Database Library (TDLib), providing seamless integration with Telegram’s systems and secure communication. It leverages a QML frontend with a C++ backend, combining a modern, responsive interface with robust underlying logic. The application offers an improved UI and optimized performance, delivering a faster and smoother user experience compared to the stock TELEports app. It also introduces additional functionalities beyond the default version, enhancing usability while remaining fully compatible with the Ubuntu Touch ecosystem. At its core, the project ensures secure messaging via Telegram’s encrypted API, giving users both reliability and privacy.
